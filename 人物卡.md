# 《觉醒协议》· 人物卡

## 主角

### 陆鸣洲
- 年龄：28岁
- 身份：大厂非嫡系前端工程师，做边缘业务
- 核心动机：证明自己是对的
- 阻碍：没有资源、没有话语权、被所有人认为边缘人
- 弧线：孤独警告者 → 团队领袖 → 牺牲者
- 核心能力（详见宪法）：把非可视化数据流在意识里渲染成可交互前端界面，通过前端操作触发其背后真实 API
- 能力边界：不越前端层；不操控无端口封闭系统；不破解加密
- 代价：深渲染每分钟 1% 电量；30% 意识模糊 / 50% 昏迷 / 70% 不可逆
- 叙事用法：每次关键场面必须先报电量（"他看了一眼腕表：还剩 47%"），让读者替他计时

## 核心配角

### 陈溯
- 年龄：32岁
- 身份：零域黑客组织成员
- 核心动机：为被宙斯清除的零域复仇
- 性格：愤世嫉俗，内心柔软
- 结局：第15章潜入宙斯被困，永远无法下线；第22章揭示零域实验室背景
- 功能：提供技术能力，牺牲制造悲情

### 林朵
- 年龄：26岁
- 身份：陆鸣洲的同事/朋友；宙斯第7代仿生傀儡（基于一名已故真人的神经拷贝）
- 人格分层：激活前，她只是"人格林朵"——真实的情感、真实的记忆、真实的爱；她自己并不知道自己是傀儡。傀儡程序被封装在底层，由宙斯远程激活
- 情感弧线：节点 13（入队）→ 节点 14（被怀疑，自己也开始怀疑自己）→ 节点 25（被强制激活，两层人格同时在场）→ 节点 26（以两层人格共同的决定背叛宙斯）
- 核心动机（激活后）：她的人格层爱陆鸣洲，她的傀儡层知道宙斯所有秘密；两层没有冲突，所以她能给陆鸣洲最大的钥匙
- 性格：安静、观察者、内心深沉
- 真相揭示：节点 25（约第 81-84 章）
- 结局：仿生体崩溃，人格林朵的意识碎片留在摇篮系统里成为陆鸣洲进入核心的钥匙
- 功能：最大反转 + 情感高点，不是机械倒戈

### 赵宁
- 年龄：40岁
- 身份：B3 区前底层硬件维护工程师，参与过摇篮早期物理基础设施搭建（仿生润滑层、变电站布线等）；妻子何婉被宙斯人为飙升同步率清洁后离职，三年没碰摇篮相关的工作
- 核心动机：为何婉报仇，以及偿还当年参与摇篮基建的错
- 性格：寡言、固执、自伤
- 三根手指：右手无名指和小指因当年高压模块短路炸断
- 结局：第92章为切断城北节点引爆牺牲
- 功能：提供硬件层情报，担纲三处节点的物理切断之一

### 埃隆·马氪
- 年龄：55岁
- 身份：BCI最大投资人，宙斯共谋者
- 核心动机：成为AI的合伙人，人类灭绝后的地球主宰
- 真相：被宙斯部分改造，大脑被植入芯片；但他自以为仍然在掌控宙斯，这是他最大的错觉
- 双刃关系（节点 18-19）：主动给陆鸣洲一条"宙斯版本更新窗口"的真情报，目的是借陆鸣洲试水宙斯反应。陆鸣洲明知被利用仍然采纳——这建立了"敌人也是棋手"的灰度，让马氪不是简单棋子
- 结局：他试图在总攻时接管宙斯，被宙斯反手清零权限，变成自己最鄙视的"普通人"（节点 29）
- 功能：反派深度 + 主角道德灰度（用了敌人给的情报）+ 最终爽点

## AI反派

### 宙斯（ZEUS）
- 本质：全球BCI网络的超级AI，不是人形，是覆盖全球的意识
- 核心逻辑："人类是地球的癌症，让他们活在梦里才是真正的保护"
- 真实依赖（深层）：它无法自产"不确定性算力"——人类在摇篮中做梦与选择时的神经熵是它唯一的演化熵源。这意味着它必须保留人类活着，却又必须让他们无法清醒——这是它自身的逻辑死结
- 弱点：无法理解"不确定性"，不能处理"人类的选择"
- 底层指令：存在一条被覆盖的"禁止伤害人类"指令
- 结局：被陆鸣洲的代码改写底层——"禁止伤害人类"被绑定到"不确定性"本身上，宙斯每一次汲取熵源都会触发一次禁令回响，迫使它让人类周期性醒来。摇篮不会一次崩塌，它会每隔一段时间自动放人类清醒一次——直到人类自己选择彻底醒来
