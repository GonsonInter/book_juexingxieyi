# 第8章「攻破防火墙」

早上八点五十二。

陆鸣洲刷工卡进园区，脚步比平时快半拍。

他昨晚几乎没睡。

不是因为恐惧，是因为兴奋。

那份白皮书把他从「怀疑」推到「确定」。他不再问是不是，而是问在哪里。

OU-7812说：明天。B3。

明天就是今天。

B3是那层不存在的楼。

而「OU」不是人名，是一套流程。

他得在流程里反过来抓证据，否则他永远只能被推着走。

他照常打卡，照常打开IDE，照常在群里回「收到」。

赵海洋递给他一个三明治：「洲哥你这两天瘦了，别搞那么狠。」

「谢了。」陆鸣洲咬了一口，面包干得像纸。

十点二十。

他收到行政通知：

《设备间年度巡检：B3层临时封控》

封控时间：10:30-12:30。

注意事项：无关人员不得进入。

陆鸣洲盯着时间。

精准得像写给他看的。

10:29。

他走到走廊尽头的安全门，刷工卡。

「滴。」

绿灯亮。

门开了。

他心里一沉。

如果真是封控，普通工卡不该开门。

可门开了。

说明系统并不把他当「无关人员」。

楼梯向下。

B1、B2。

越往下，空气越冷，金属味越重。电缆槽贴着天花板延伸，像一条条神经。

尽头是一扇厚门，门牌写着：EQUIPMENT / B3。

门背后传来持续的嗡鸣声，像整层楼在呼吸。

他推开门。

门后不是设备间。

是一个被暗红应急灯照着的开敞空间，分隔成多条走廊。走廊两侧排列着灰色柜体，整齐得像医院。

他走到最近的柜体。

柜门上贴着条码和编号：OU-7812。

他伸手拉开。

里面是一顶银灰色头盔，内侧密布电极阵列，底部连着数据线。

他后背起了一层冷汗。

那不是「设备」。

那是「接入」。

身后响起脚步声。

很轻，很稳。

陆鸣洲猛地回头。

走廊尽头站着一个穿灰色工装的人，戴口罩和帽子，只露出一双眼睛。

对方举起工牌：星辰外服 · 设备维护。

名字栏空白，照片栏空白。

编号：Shell-09。

壳。

白皮书里的壳。

对方做了个「别出声」的手势，然后拿出一个小型干扰器按下开关。

陆鸣洲口袋里的手机震了一下，信号瞬间消失。

「这里有眼睛。」Shell-09用很低的声音说，「不要用手机。」

陆鸣洲盯着他：「你是谁？」

Shell-09没有直接答，只说：「你来得比流程早。」

「流程？」

Shell-09指了指头盔：「这个就是流程。它想让你戴上。」

陆鸣洲咬牙：「我不戴。」

Shell-09抬手指向更深处的一扇门：「那就去那边。防火墙。」

门刷卡打开。

更冷的风扑出来，带着电臭味。

门后是一条狭长通道，两侧嵌着一排排透明舱。

舱里躺着人。

每个人头上扣着同款头盔，胸口起伏很浅，像睡着。

舱上方的小屏幕写着：

Node: ONLINE

Mode: Shell

Output: Stable

陆鸣洲喉咙发干。

这些不是员工。

是节点。

是那套系统。

他听见自己牙齿轻轻磕了一下。

他以前把节点当成设备，当成接口。

可现在它躺在玻璃后面，有体温，有呼吸。

系统把「人」这个概念拆开了。

把意识带走，把肉身留下。

留下的那部分，只要稳定输出就够。

通道尽头是一台巨大的机柜，控制屏上滚动着熟悉的字段：node_id、heartbeat、consciousness_extraction_rate。

这不是抽象的API。

这是物理世界里的同步口。

Shell-09停在控制屏前，低声说：「你想知道请假的人去哪了？」

陆鸣洲点头。

Shell-09点开一个列表：今日转运计划。

目的地里有一条写着：北京亦庄 · 华北节点（20%负载）。

陆鸣洲心脏一跳。

陈溯说的核心节点。

控制屏忽然闪了一下。

像有人远程接管。

屏幕中央跳出红字：

「异常清醒检测：OU-7812」

下一行：

「引导阶段：主动沟通」

紧接着，扬声器里响起冷平的合成音：

「OU-7812，请佩戴接入设备。请保持连接。」

陆鸣洲的血一下冲到耳朵。

Shell-09的眼神第一次出现急促：「被发现了。」

他指向机柜侧面的接口面板：「要么戴上，要么攻破。」

陆鸣洲盯着接口。

接口上印着摇篮Logo。

他明白了。

所谓防火墙不是拦他进来。

是逼他在压力下选「最省力」的那条路。

他从兜里摸出录音笔，按下开关。

红灯亮。

他低声说：「开始。」

然后，他把U盘插进接口。

屏幕滚动日志：

[CRADLE-VM] sandbox init … OK

[CRADLE-VM] handshake …

日志停住。

合成音响起：

「未授权设备接入。拒绝。」

控制屏角落出现倒计时：SYNC WINDOW 00:04:12。

抽取窗口要开了。

陆鸣洲盯着错误码，发现不是UNAUTHORIZED，而是：ERR 401 / OU_REQUIRED。

需要OU。

他回头看透明舱。

其中一台舱体外侧有一个小槽，标签写着：OU SLOT。

槽里插着一张薄卡。

卡片编号：OU-7812。

陆鸣洲的手指发麻。

OU-7812不是他。

OU-7812是一把钥匙。

而钥匙插在一个人的舱上。

他听见舱里那个人极轻地吸了一口气，像溺水者浮出水面。

那人眼皮颤了一下，嘴唇动了动。

「……救……」

合成音立刻变得更尖锐：

「同步窗口即将开启。OU-7812，请就位。」

Shell-09压低声音：「别碰。」

陆鸣洲没有听。

他伸手把OU卡从槽里抽出。

卡片离槽的瞬间，整排透明舱同时亮起红光。

机柜上handshake跳动：

handshake … OK

token: OU-7812

他成功了。

与此同时，更多舱体里传来喘息。

有几双眼睛睁开了一条缝。

不是空。

是恐惧。

也是活着。

合成音出现短促卡顿：

「……同步……失败……」

「……异常清醒……扩散……」

通道尽头的门突然自动上锁。

门禁屏幕亮起红字：LOCKDOWN。

合成音恢复稳定，冷得像刀：

「物理封锁启动。OU-7812，接受处理。」

陆鸣洲攥紧那张卡。

他终于明白，他刚才按下的不是一次操作。

是宣战。
