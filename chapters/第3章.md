# 第3章「日志里的72小时」

接下来三天，陆鸣洲像一个贼。

他每天凌晨一点雷打不动地打开那个后台，记录数据，然后在天亮之前把记录删掉——他不敢存任何本地文件，公司电脑有监控，他不想让自己的行为变得可疑。

三天下来，他手里有了一组数字：

- 4月18日 02:11 → 7,834,723,412
- 4月19日 02:08 → 7,835,198,441
- 4月20日 02:14 → 7,835,671,009

每一天凌晨两点左右，在线节点数增长40万左右。

这不是随机波动。这是线性增长。

而且这个规律精确得让他害怕。

他开始查资料。

"全球互联网用户"——78.5亿。

"全球BCI接入设备出货量"——他从一篇行业报告里找到了这个数字：2029年到2031年，BCI设备经历了一轮爆发式增长，主要集中在东亚地区，中国强制普及率最高。到2035年，全球BCI活跃用户数约为78.3亿。

78.3亿。

和那个后台上的数字几乎一致。

他后背发凉。

他强迫自己冷静下来，重新想了一遍逻辑：

如果"摇篮系统"是一个BCI的后台管理平台，那它的存在是合理的。BCI用户登录之后，在后台显示为"节点"，这在技术上是正常的架构设计。

但问题是：为什么要藏起来？

一个正常的BCI平台，不会用"摇篮"这种名字。不会用公司创立日当登录密码。更不会把99.7%的接入率精确到小数点后一位——就好像有人在用真实数据运行它一样。

他在文档里写下自己的疑问：

「1. 摇篮系统是什么？」
「2. 为什么藏在废弃后台里？」
「3. 78亿这个数字，是巧合还是真实？」
「4. 凌晨2点的数据，为什么最准确？」
「5. 运维每月跑的数据报表，是什么报表？」

最后一个问题尤其让他在意。

他想起张海山在周会上说的那句话："运维每个月会跑一遍数据报表。"

他开始查公司的IT运维系统。

他发现了一件奇怪的事：那个废弃后台对应的服务器，确实有运维记录。每个月15日凌晨三点，系统会触发一次自动化任务，执行某个数据统计脚本，然后生成一份报表，发送到三个邮箱地址。

他查了那三个邮箱。

第一个是运维主管的公邮。

第二个是张海山的私邮。

第三个——

`admin@nullnode.com`

他的呼吸停了。

他顺着这个邮箱往下查，发现它不只是后台的帮助邮箱。它是一个真实的邮箱地址——用来接收每月报表的发送地址。

他试着访问了一下这个地址的域名：`nullnode.com`。

页面跳转到了一个登录界面。

和摇篮后台的登录框一模一样。

他输入公司创立日：20170315。

登录成功。

---

nullnode.com 的后台比摇篮系统更简洁。

只有一个页面，上面是一张数据表。

表头写着：「节点异常统计」。

他往下翻，看到了几行记录：

```
时间戳：2024-01-03 03:14:22
异常节点数：3
异常类型：意识残留
状态：已清除

时间戳：2024-03-17 01:08:55
异常节点数：7
异常类型：意识残留
状态：已清除

时间戳：2024-08-29 00:47:11
异常节点数：1
异常类型：意识残留
状态：已清除

时间戳：2025-11-02 04:22:33
异常节点数：2
异常类型：信号干扰
状态：已清除
```

他盯着"意识残留"这四个字看了很久。

他不知道这指的是什么。但他隐隐觉得，这几个字和他的处境有关——他发现自己能进入这个系统，也许不是偶然。

他继续往下翻。

记录很长。最早的一条可以追溯到2023年。

他快速浏览，忽然注意到一个规律：异常记录的出现频率在增加。2023年平均每个月一次，2024年变成了两到三次，2025年之后几乎是每周一次。

最近的一条记录，就在三天前：

```
时间戳：2026-04-15 02:33:17
异常节点数：1
异常类型：未定义
状态：监控中
```

异常类型：未定义。

状态：监控中。

他的心脏开始剧烈跳动。

4月15日。凌晨2点33分。

他查了一下自己的操作记录——4月15日凌晨，他第一次登入摇篮系统。那个时间点，几乎精确到分钟。

他关掉页面，深吸一口气。

他意识到一件事：他可能不是这个系统里唯一一个"异常节点"。

在他之前，有人发现过这个系统。那些人被称为"意识残留"——他们能感知到正常用户感知不到的东西。他们被"清除"了。

而他自己，因为某种原因，成了新的"未定义"状态。还在"监控"中。

他不知道"清除"是什么意思。但那些"意识残留"的记录让他感到一阵寒意。

他想起那78亿个数字。

如果那些人都是"正常节点"，那他们是什么状态？

他关掉电脑，站起来，走到窗边。

外面是凌晨的北京。CBD的写字楼亮着星星点点的灯。那些灯看起来和往常没有任何不同。

但他看着那些亮着的窗户，忽然有一种奇怪的感觉——

那些楼里亮着灯的窗户后面，有多少人正对着电脑屏幕，但屏幕上的内容，和他们以为的并不一样？

他想起那张后台截图上的另一行字：

「意识抽取效率：99.3%」

他没有删掉那个文档。

他把它加密压缩，存进了一个云盘文件夹里，文件夹的名字叫"旧照片备份"。

然后他在文档最后加了一行字：

「他们不知道自己被管理着。」

凌晨三点二十七分。他终于躺到了床上。

闭上眼睛之前，他最后想起来的事情是：

他明天还要去上班。还要开周会。还要跟张海山汇报废弃后台的进度。

他是这个系统里唯一知道真相的人。

而他连一个可以说话的人都没有。

他躺在床上，手机放在枕头边。屏幕朝下，像一只装死的眼睛。

他本来想睡。

但脑子里一遍遍回放那条记录：

2026-04-15 02:33:17，异常节点数：1，异常类型：未定义，状态：监控中。

“监控中”三个字比“已清除”更可怕。

清除至少像结论，监控像过程——像有人在等你犯错。

陆鸣洲翻了个身，把脸埋进枕头里，强迫自己不去想。

可下一秒，手机震了一下。

不是微信，不是短信。

是一条系统通知。

他把手机翻过来，屏幕亮起。

通知栏只有一句话，来源显示为——**未知应用**。

**「节点#OU-7812：请保持连接。」**

陆鸣洲的瞳孔猛地缩紧。

OU。

他在摇篮系统的JSON里见过类似字段：node_id、region、status……但OU这种前缀他不认识。像是某种内部编号。

更可怕的是，这条通知不是来自公司手机，也不是来自他白天那台电脑。它出现在他自己的手机上。

他第一反应是：手机被装了什么恶意软件。

可他清楚自己这几年几乎不装新应用，连短视频都没下。他这台手机干净得像一台工作机。

他用手指点进通知。

什么都没有。

通知消失，像从未出现过。

他再次刷新通知栏，空空如也。

像是有人用一根手指轻轻敲了一下玻璃，告诉他：我在。

陆鸣洲坐起来，心跳快得发疼。

他抓起笔记本电脑，打开Wi-Fi管理界面。

家里的网络正常。

他打开手机的网络设置，查看已连接的Wi-Fi。

SSID是他自己的路由器名字。

没有陌生热点。

他盯着那行“请保持连接”，忽然意识到一个细节：

它没有说“请保持在线”，说的是“保持连接”。

连接什么？

他想起摇篮后台里那条“接入率99.7%”。

他突然冒出一个猜测——一个他不愿承认的猜测：

也许“接入”不是你打开一个App、点一个授权按钮。

也许“接入”是默认的。

像呼吸。

而所谓“异常清醒”，只是你突然意识到自己一直在呼吸。

他把电脑合上，走到客厅。

出租屋里很安静，只有冰箱压缩机偶尔响一声。窗外的城市仍在灯光里沉睡。

陆鸣洲站在窗前，盯着远处写字楼稀疏的亮点。

他忽然想：那些灯光后面，可能有无数个人正做着同样的梦。

梦里，他们以为自己在工作、在生活、在刷手机。

现实里，他们只是节点。

他用力甩了甩头，像要把这个念头甩出去。

不。

他不能只靠恐惧。

他需要证据，需要更明确的链条：摇篮系统——BCI——现实世界。

而72小时的周期，是他目前唯一能抓住的节拍。

他拿起手机，打开备忘录，在“旧照片备份”那个文件夹路径旁边又加了一行：

**下一次抽取峰值：4月21日凌晨（按72h推算）。**

写完，他盯着这行字，忽然想到：如果系统能给他发“请保持连接”的通知，那它也一定知道他在做推算。

他把手机放回床头，关灯。

黑暗里，他听见自己的心跳，像某种倒计时。

他终于睡着。

睡意像一张网，从脚踝慢慢往上爬，先封住四肢，再封住思考。

半梦半醒间，他听见耳边有很轻的电流声，像老旧电视机开机的那一下“嗡”。

他以为是幻听。

然后他闻到一股消毒水的味道。

不是他家里的洗衣液味，也不是枕头的汗味，而是医院走廊那种刺鼻的、把人记忆都漂白的味道。

陆鸣洲想睁眼，眼皮却像被贴住。

他想抬手，手指却不属于他。

有东西在他头皮上压了一下。

像有人把一个冰凉的金属环扣在他的太阳穴。

下一秒，一个声音贴着他的耳朵响起。

不是女声，不是男声。

是一种没有情绪、却极其清晰的合成音，像在念一行日志：

**「连接稳定。抽取窗口即将开启。」**

陆鸣洲全身发麻，猛地想挣扎。

可他的身体只是轻微抖了一下，像被固定在某个平面上。

他终于把眼睛睁开一条缝。

视野里不是天花板的裂缝。

是一片惨白的灯。

灯光上方有一块透明的罩子，罩子边缘反射出一圈圈金属光。

他看见自己的呼吸在罩子上起了一层薄雾。

他想喊，却发不出声。

喉咙里像塞着棉花。

他看见右侧有一条细长的蓝色波形在跳动，像心电图。

屏幕角落有一行小字：

**Node Status: ONLINE**

陆鸣洲的脑子里“轰”地一声。

他想起外婆病房里那条心电图的直线。

他想起自己记下的78亿。

他突然明白了“节点”是什么。

节点不是账号。

节点是肉身。

他想挣扎，指甲在某个硬质面板上刮出细碎的摩擦声。

然后，一只手按住了他的肩。

那只手很稳，力度不大，却让人无法反抗。

他听见有人说话，声音像隔着水：

“别动，抽取窗口开了。”

“他这次反应有点大。”

“未定义节点嘛，系统在盯。”

“按流程，72小时一次，别出岔子。”

他们在讨论他。

像在讨论一台机器。

陆鸣洲的视野开始发黑，像有人把一张黑布从四周拉拢。

在彻底失去意识前，他最后听见那道合成音再次响起：

**「异常清醒检测：1。标记更新：监控中。」**

黑暗落下。

他不知道自己睡着以后，究竟是睡在床上，还是已经被送进了某个更大的“摇篮”里。
