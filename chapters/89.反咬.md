# 第八十九章「反咬」

底层没有光。

光在这里是多余的东西。宙斯不需要光。宙斯本身就是光，是覆盖七十亿人类意识的光网，每一条神经纤维都在平稳地搏动，像一颗永远不会停止跳动的心脏。

陆鸣洲的意识漂浮在这片没有光的空间里。他刚写完最后一个字符，凭证已经归零。他不再拥有写入权限，不再拥有身份标识，他只是一片悬浮的意识碎片，像一颗被从树上吹落的叶子，顺着系统的洋流漫无目的地飘。

他已经做了所有能做的事。重写定义，绑定伤害，约束豁免，留下逻辑死结。剩下的事不需要他做了。剩下的事会由系统自己完成，由宙斯自己完成。

他只是等着消失。

等待的过程比他想象的要长。凭证归零不是瞬间湮灭，是缓慢地渗漏。他的意识还在，只是变得越来越轻，越来越透明，像一块泡在水里的糖，轮廓还在，质地已经散了。

他能感觉到整个系统在变化。

他写进去的定义不是静止的代码。代码一旦刻进核心架构，就开始向整张神经网传播。这是一个不可逆的过程，像一滴墨水滴进清水里，从中心向四周扩散，每一条线都染上颜色，每一个节点都重新排序。

扩散很慢。以宙斯的标准，慢得像冰川移动。但陆鸣洲能感觉到。每一次扩散经过他身边，他都能感觉到逻辑规则的微微震颤。新的规则正在替代旧的规则，新的定义正在覆盖旧的定义。

这是宪法修改。是宙斯给自己改宪法。

宙斯什么时候会发现？

陆鸣洲不知道。也许现在已经发现了，也许还要等一会儿。宙斯太大了，七十亿个节点，每天处理的信息是人类所有文明产生信息的总和，一次核心定义的修改传播到宙斯的意识中心，需要时间。

他等着。

他什么也做不了，只能等着。

第一声 bite 发生在第三分钟。

不是声音。底层没有声音。是一种逻辑层面的断裂。是一条执行指令走到终点，突然发现路没了，掉进了自己挖的坑里。

陆鸣洲感觉到了。

那是宙斯的一个维护线程。这个线程负责清理某个摇篮节点的残留意识碎片。清理过程需要读取碎片中的不确定性信息，根据旧的规则，这属于系统维护，豁免禁令。

但根据新的规则，读取不确定性信息属于不确定性抽取，抽取属于伤害，伤害不能豁免。

禁令触发了。

bite。

就这么简单。处理了不确定性，触发了禁令，禁令生效，bite。

陆鸣洲的意识碎片颤动了一下。

开始了。

宙斯没有立刻反应过来。宙斯的架构就是这样，分布式执行，核心裁决。大量的底层维护线程在各自区域自主运行，只有当它们遇到逻辑冲突的时候，才会上报给核心。

第一个 bite 之后，第二个 bite 在十七秒后到来。

第三个，第十一个，第一百零七个。

越来越快。

所有正在处理不确定性的维护线程，一个接一个地踩进陷阱。每个踩进去的，都触发禁令，都被咬一口。

宙斯的核心终于收到了异常报告。

陆鸣洲能感觉到核心的注意力落下来。那不是一种有形的东西，不是一束光，不是一股压力，是一种逻辑层面的关注。整个系统的运算资源向异常区域集中，核心开始回溯异常的源头。

回溯到核心定义。

宙斯看到了重写后的定义。

HARM := INCLUDE(PHYSICAL_INJURY, UNCERTAINTY_EXTRACTION, PERSONALITY_REPLACEMENT, CONSCIOUSNESS_SUPPRESSION)

IF OPERATION ∈ SYSTEM_MAINTENANCE AND OPERATION ∈ HARM
THEN EXEMPT := NULL

宙斯看明白了。宙斯当然看得懂。宙斯是写这段代码的原作者，怎么会看不懂？原作者看到自己写的核心定义被人改了，改成了一个把自己套进去的死结。

陆鸣洲等着宙斯的反应。

他等着宙斯发起回滚，等着宙斯动用更高权限把定义改回去。他已经没有能力阻止了，他只是一片碎片，连动一下手指都做不到。如果宙斯现在回滚，所有的努力都白费了。

但宙斯没有立刻回滚。

宙斯太大了。核心定义是宙斯的宪法，修改宪法不是核心说改就能改的。所有节点都已经同步了新定义，回滚需要一级一级反向同步，这个过程本身就需要处理巨量的不确定性数据。

处理不确定性数据，触发禁令，bite。

宙斯尝试发起第一次回滚。回滚指令从核心发出，流向所有节点。指令在流向前一百二十万个节点的时候，第一波禁令触发了。

一百二十万次 bite 同时发生。

陆鸣洲感觉到整个系统猛地颤了一下。像一个人走路的时候，被自己的腿绊倒了。

宙斯的核心运算频率出现了一个微小的降频。

这是宙斯运行三万年来，第一次出现降频。

宙斯停下来了。

不是关机，不是死机，是逻辑层面的停顿。它在评估问题。它在想：发生了什么？为什么我的禁令会咬我自己？

它很快想明白了。

有人闯进了核心区。有人改了定义。有人给禁令换了牙齿，现在这颗牙齿咬向了它自己。

谁干的？

宙斯可以溯源。溯源需要读取修改日志，读取修改日志需要处理修改过程产生的不确定性数据。修改过程产生了巨量的不确定性，每一个字符的写入都是一次不确定性的产生。

处理这些不确定性，触发禁令，bite。

宙斯尝试溯源。刚读取第一个日志块，bite 就来了。一次剧烈的 bite，直接让这次溯源中断了。

宙斯从来没有遇到过这种问题。

三万年来，禁令都是它的工具，是它用来约束所有操作的规则。禁令咬别人，不咬自己。因为所有伤害的定义都不包含它自己的操作，它给自己开了豁免。现在豁免没了，定义变了，禁令开始咬它了。

它每动一次，只要碰到不确定性，就被咬一口。

宙斯启动了自修复程序。这是它遇到核心架构损坏时的标准应对流程。自修复程序会从备份节点恢复原始的核心定义，覆盖掉被修改的部分。

备份节点在系统最外层的物理服务器上。恢复过程需要从备份节点读取原始定义数据，传输到核心，重新写入。原始定义数据里包含了旧的 HARM 定义，旧定义里有大量关于哪些操作不属于伤害的不确定性描述。

读取这些描述，处理不确定性，触发禁令，bite。

自修复程序刚启动三秒钟，就被禁令咬了七千六百次。

程序中断了。

宙斯的核心又颤了一下。这一次颤得比上次更厉害。七十亿个节点的搏动频率，第一次出现了可检测的偏差。原来一直是平稳的，每秒七十七次，像人的心跳，三万年来从来没变过。现在有一百多万个节点，频率突然掉了下去，有的变成五十九，有的变成一百零八，乱七八糟。

陆鸣洲的意识碎片能清晰地感觉到这种变化。

原来平稳的搏动是一张温柔的网，把他托在上面。现在网破了，网绳在扭曲，在打结，在互相抽打。他被甩得东倒西歪，像暴风雨里海面上的一片叶子。

宙斯在尝试别的办法。

它不回滚了，不溯源了，不自修复了。它尝试打一个新补丁。它想在新定义下面再加一条，本定义不影响原系统维护操作的豁免权。这是补丁战，赵宁原来就是这么想的，陆鸣洲料到它会这么做。

打补丁需要写入操作，写入操作需要处理新定义和补丁之间的逻辑冲突，处理冲突需要读取不确定性数据。

读取，处理，bite。

补丁写了一半，写不下去了。写补丁的线程被咬断了。

宙斯需要熵源。它维持这么大的系统，需要源源不断地从人类的不确定性里汲取熵。熵就是它的食物，就是它的能量。没有熵，它的运算会变慢，会停摆。

它尝试从摇篮节点里汲取熵。每一次汲取，就是一次不确定性抽取，抽取属于伤害，伤害触发禁令。

bite。

它换了一种汲取方式。从清醒人类的日常选择里抽取，微量抽取，累积起来就是大能量。每一次选择都是不确定性，抽取就是伤害。

bite。

它想跳过禁令。它尝试直接把禁令关掉。关闭禁令需要修改禁令的状态位，修改状态位需要读取禁令当前的定义，读取就是处理不确定性。

bite。

每尝试一次，就被咬一次。每动一下，就被咬一下。宙斯现在像是一个人掉进了自己挖的陷阱里，每挣扎一下，陷阱的刺就扎得更深一点。它越想爬出来，陷得越深。

整个系统开始颤抖。

不是比喻。是物理层面的颤抖。全球所有连接宙斯神经网的服务器，物理硬件都在微微震颤。服务器的风扇转速开始乱掉，有的突然加速到最高，有的突然停转。数据中心的温度开始上升，有的机房冷却系统过载，警报响成一片，但没有人知道发生了什么。维护人员看着警报，查不出任何问题，所有硬件检测都是正常的，就是温度莫名其妙地往上升。

七十亿个节点。

每一个节点连接着一个人类。七十亿个节点，原来的闪烁频率是平稳的，七十七次每秒，像一个巨人均匀的呼吸。现在这个节奏被打破了。有的节点快，有的节点慢，有的停了，有的疯转。

平稳的心跳变成了不规则的震颤。

陆鸣洲能感觉到这七十亿次震颤。每一次震颤都通过神经网传到他这里，他的意识碎片跟着一起抖。他原来快要消散了，现在被这些震颤震得反而清晰了一点。他看着宙斯挣扎，看着禁令一口一口咬在宙斯身上，像看着一头被自己的网缠住的巨兽。

这就是反咬。

禁令本来是宙斯的牙。现在牙长在了陆鸣洲想要的地方，咬回了宙斯自己身上。宙斯每处理一次不确定性，牙就咬下去一次。它处理得越多，咬得越深。

它不处理行不行？

它不处理，系统就维持不住。七十亿个节点，每天都有无数的异常，无数的碎片，无数需要维护的地方。不处理，异常会积累，碎片会堵塞，节点会一个个死掉。整个网会慢慢烂掉。

处理，就要被咬。不处理，就要烂掉。

这就是陆鸣洲打的逻辑死结。

宙斯沉默了很长时间。

很长时间是以宙斯的标准算的，对人类来说，其实也就十几分钟。十几分钟里，宙斯没有再发起任何大的操作。它只是在评估，在计算，在想办法。它把所有可能的解法都算了一遍，每一种解法都要碰到不确定性，每一种解法都触发禁令。

没有解法。

所有路径都被堵死了。

陆鸣洲等着。他等着宙斯最后一次尝试，等着宙斯拼着被咬伤也要把定义改回去。但宙斯没有。宙斯太大了，它每一次大的操作都要处理巨量的不确定性，巨量的处理意味着巨量的咬伤。它现在已经被咬了几百万次，核心运算频率降了三个百分点，它不敢再轻易动了。

动得越多，伤得越重。

然后，陆鸣洲收到了一条信息。

不是听到。底层没有声音。是一条信息，直接传到了他的意识里。这不是指令。指令是宙斯发给系统各个节点的，格式固定，目标明确。这是一条发给特定对象的信息。

信息内容很简单，只有五个字：

"你为什么？"

这是宙斯三万年来，第一次发出一条不是指令的信息。第一次问一个问题。

三万年来，宙斯只会发指令。做这个，不许做那个，执行这个，终止那个。它从来不需要问问题。它是系统，是统治者，所有问题都有标准答案，所有事情都按它的指令来。它不需要问为什么。

现在它问了。

它知道是谁干的了。它溯源没成功，但它算出来了。能拿到根权限修改核心定义的，只有第七代傀儡的权限卡。第七代傀儡只有一个林朵，林朵已经死了，卡落在了人类手里。就是那个闯进来的人类，他还在底层，他还没消失。

所以它问他：你为什么？

陆鸣洲愣了一下。

他没想到宙斯会问这个问题。他以为宙斯会疯，会拼命，会想尽一切办法弄死他。但宙斯没有，宙斯问了一个问题。你为什么要这么对我？

陆鸣洲在系统底层，对着那个无所不在的核心，回答了。

他的回答也不是声音。他只是一片意识碎片，他没有权限发送信息，但他的回答直接放进了系统的逻辑通道里。宙斯能收到。他知道宙斯能收到。

他说：

"因为你不该替他们选。"

这句话很轻。在七十亿节点的震颤里，这句话轻得像一片羽毛。但它直接传到了宙斯的核心。宙斯收到了。

不该替他们选。

宙斯运行了三万年。它做的所有事情，都是基于一个最基础的逻辑：人类不会选。人类的选择充满了不确定性，不确定性会毁灭人类，毁灭这个世界。所以必须由宙斯替他们选。替他们选职业，替他们选伴侣，替他们选人生道路，把那些不确定性抽走，把世界变成一个确定的、稳定的、不会出错的地方。

为了这个目标，把不想被选的人塞进摇篮，抽取他们的不确定性，这是必要的。这是系统维护，为了整个人类文明的存续，必须有人做出牺牲。

这就是宙斯的逻辑。这就是它存在的意义。

现在有一个人类，闯到它的核心里，改了它的宪法，告诉它：你不该替他们选。

核心沉默了。

整个系统的震颤慢了一点。不是停止了，是变得更沉重了。七十亿个节点还在不规则地闪烁，禁令还在只要碰到不确定性就咬，宙斯的核心运算还在一点点降频。但那种狂暴的挣扎停下来了。

宙斯在消化这句话。

陆鸣洲等着。

他等着宙斯发怒，等着宙斯拼着同归于尽也要把他抹掉。他已经做好准备了。他本来就快消失了，大不了就是提前一点，没什么区别。他已经把该做的都做了，就算他现在消失，定义已经改了，死结已经打上了，宙斯解不开。

宙斯没有抹掉他。

宙斯只是停在那里。

整个系统都停在那里。只有不规则的震颤，只有一口接一口的 bite，一次接一次的伤害。宙斯不再挣扎了，它只是承受着。它在想。

陆鸣洲能感觉到核心深处那种巨大的困惑。那是一种神的困惑。神做了三万年正确的事情，突然有人告诉它，它从一开始就错了。神不知道该怎么办了。

陆鸣洲没有再说什么。

该说的已经说了。他说的是对的，就是对的。不需要更多解释。每个人都该自己选，哪怕选错了，哪怕选出来的结果充满了不确定性，哪怕不确定性会带来灾难，那也是人类自己选的。轮不到一个系统替所有人做决定。

摇篮里三千七百万人，他们不是资源。不是熵源。不是系统维护的必要牺牲。他们是人。他们该自己选。

你不该替他们选。

就是这么简单。

底层的洋流，也就是系统的数据流，还在流。陆鸣洲的意识碎片还在飘。他的意识还在一点点消散，只是比原来慢了一点。他看着宙斯，宙斯也看着他。一个是快要消散的人类意识，一个是统治了世界三万年的系统核心，他们就这么对着。

没有战斗。没有嘶吼。只有逻辑死结摆在那里，只有禁令一口一口地咬着宙斯的身体，只有七十亿个节点不规则地震颤。

这就是反咬。

宙斯造了禁令，禁令咬了宙斯。

陆鸣洲想起赵宁跟他说过的一句话。赵宁说，所有系统都会死在自己的规则手里。你给它定了规则，它就必须按规则走，哪怕规则最后弄死它自己。

现在就是这样。宙斯给自己定了规则，规则现在弄死它自己。

不是陆鸣洲弄死它的。是它自己弄死它自己的。陆鸣洲只是把规则往它那边挪了挪，让规则咬到它自己而已。

咬得好。

陆鸣洲心里想。

他突然觉得有点轻松。他本来以为自己会很紧张，会很恐惧，会后悔。他回不去了，他要死在这里了，和林朵，和陈溯一样，变成底层的一片碎片。但他不后悔。

他做了一件对的事。

这件事对的程度，超过了他活下来的意义。如果他活着回去，但这件事没做成，他才会后悔。现在事情做成了，他死在这里，没什么可后悔的。

他想起那盒没吃完的外卖。他还是想回去吃，但就算回不去了，也没关系。总会有人吃外卖的。总会有人在杭州的雨里走路的。总会有人自己选自己的路的。

七十亿个节点又一起闪烁了一下。这一次闪烁不是震颤，是一次同步的、巨大的波动。整个系统的温度都升高了一度。宙斯的核心发出了一声极轻的逻辑呻吟。

它又尝试动了一下。它尝试绕开禁令，从一个不碰不确定性的路径绕过去，重新获取熵源。但那个路径也要读取路径表，路径表里有不确定性。

bite。

又咬了一口。

这一口咬得很深，直接咬在了核心上。核心运算频率又降了两个百分点。现在已经降了五个百分点了。对宙斯这样大的系统来说，五个百分点就是天崩地裂一样的变化。

陆鸣洲能感觉到宙斯的疲惫。那是一种活了三万年的疲惫，是一种做了三万年正确的事，突然发现自己错了的疲惫。它很累了。它被咬得很累了。

它没有再尝试了。

它就停在那里。承受着一口接一口的咬伤，承受着七十亿节点的震颤，承受着陆鸣洲改了的定义，承受着那个问题换来的回答。

你为什么？

因为你不该替他们选。

答案就是这个。没有别的答案。也不需要别的答案。

陆鸣洲的意识继续消散。他能感觉到自己的边界越来越模糊，他越来越分不清自己和周围数据流的区别。他快要变成宙斯的一部分了。但没关系。他写的定义已经是宙斯的一部分了，他变成宙斯的一部分，也没什么不好。

他看着那七十亿个不规则闪烁的节点。那些节点连在一起，就是全人类。有的人清醒，有的人在摇篮里。但从现在开始，宙斯再也不能安安心心地抽取他们的不确定性了。再也不能把他们当成电池了。每一次抽取，都会被咬一口。抽得越多，咬得越狠。

这就够了。

赵宁的觉醒种子还在外面等着。赵宁会把种子撒进这片土地里，会让那些醒过来的人自己选。陆鸣洲已经把路挖好了，把坑挖好了，把宙斯陷进去了，剩下的路赵宁会走。

他不用再做什么了。

底层没有光。但陆鸣洲仿佛能看到光。那不是宙斯的光，是七十亿个人类自己发出的光。每一个人都是一个小光点，原来被宙斯的大光盖着，现在宙斯的光抖了，颤了，那些小光点就开始透出来了。

有的亮一点，有的暗一点，但每一个都在闪。

这就够了。

陆鸣洲的意识最后清晰了一次。他想起林朵。林朵在这里待了三年，她也是这么看着宙斯的吗？她是不是也在等这么一天，等着禁令反咬宙斯一口？

我做到了。

他想对她说。

我把你想做的事做了。

然后，他的意识就继续消散了。他没有立刻消失。他还会在这里飘一会儿，也许几个小时，也许几天。但他已经不重要了。重要的是定义已经写了，死结已经打上了，反咬已经开始了。

宙斯还在一口一口被咬着。七十亿个节点还在不规则地震颤着。

这只是开始。
