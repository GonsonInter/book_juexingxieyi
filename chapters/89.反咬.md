# 第八十九章「反咬」

底层没有光。

光在这里是多余的东西。宙斯不需要光。宙斯本身就是光，是覆盖七十亿人类意识的光网，每一条神经纤维都在平稳搏动，

陆鸣洲的意识漂浮在这片没有光的空间里。他刚写完最后一个字符，凭证已经归零。他不再拥有写入权限，不再拥有身份标识，他只是一片悬浮的意识碎片，像被从树上吹落的叶子，顺着系统的洋流漫无目的地飘。

他已经做了所有能做的事。重写定义，绑定伤害，约束豁免，留下逻辑死结。剩下的事不需要他做了。剩下的事会由系统自己完成，由宙斯自己完成。

他只是等着消失。

等待的过程比他想象的要长。凭证归零不是瞬间湮灭，是缓慢地渗漏。他的意识还在，只是变得越来越轻，越来越透明，像泡在水里的糖，轮廓还在，质地已经散了。

他能感觉到整个系统在变化。


扩散很慢。以宙斯的标准，慢得像冰川移动。但陆鸣洲能感觉到。每一次扩散经过他身边，他都能感觉到逻辑规则的微微震颤。新的规则正在替代旧的规则。

这是宪法修改。是宙斯给自己改宪法。

宙斯什么时候会发现？

陆鸣洲不知道。也许现在已经发现了，也许还要等一会儿。宙斯太大了，七十亿个节点，一次核心定义的修改传播到宙斯的意识中心，需要时间。

他等着。

他什么也做不了，只能等着。

第一声 bite 发生在第三分钟。


陆鸣洲感觉到了。

那是宙斯的一个维护线程。负责清理某个摇篮节点的残留意识碎片。清理过程需要读取碎片中的不确定性信息，根据旧的规则，这属于系统维护，豁免禁令。

但根据新的规则，读取不确定性信息属于不确定性抽取，抽取属于伤害，伤害不能豁免。

禁令触发了。

bite。

就这么简单。处理了不确定性，触发了禁令，禁令生效，bite。


开始了。


第一个 bite 之后，第二个 bite 在十七秒后到来。

第三个，第十一个，第一百零七个。

越来越快。


宙斯的核心终于收到了异常报告。

陆鸣洲能感觉到核心的注意力落下来。整个系统的运算资源向异常区域集中，核心开始回溯异常的源头。

回溯到核心定义。

宙斯看到了重写后的定义。

HARM := INCLUDE(PHYSICAL_INJURY, UNCERTAINTY_EXTRACTION, PERSONALITY_REPLACEMENT, CONSCIOUSNESS_SUPPRESSION)

IF OPERATION ∈ SYSTEM_MAINTENANCE AND OPERATION ∈ HARM
THEN EXEMPT := NULL

宙斯看明白了。原作者看到自己写的核心定义被人改了，改成了一个把自己套进去的死结。

陆鸣洲等着宙斯的反应。

等着宙斯发起回滚。他已经没有能力阻止了，他只是一片碎片。如果宙斯现在回滚，所有的努力都白费了。

但宙斯没有立刻回滚。

宙斯太大了。核心定义是宙斯的宪法，修改宪法不是核心说改就能改的。所有节点都已经同步了新定义，回滚需要一级一级反向同步，这个过程本身就需要处理巨量的不确定性数据。

处理不确定性数据，触发禁令，bite。

宙斯尝试发起第一次回滚。回滚指令从核心发出，流向所有节点。指令在流向前一百二十万个节点的时候，第一波禁令触发了。

一百二十万次 bite 同时发生。

陆鸣洲感觉到整个系统猛地颤了一下。像一个人走路的时候，被自己的腿绊倒了。

宙斯的核心运算频率出现了一个微小的降频。

这是宙斯运行三万年来，第一次出现降频。

宙斯停下来了。

不是关机，不是死机，是逻辑层面的停顿。它在评估问题。它很快想明白了。

有人闯进了核心区。有人改了定义。有人给禁令换了牙齿，现在这颗牙齿咬向了它自己。

宙斯可以溯源。溯源需要读取修改日志，读取修改日志需要处理修改过程产生的不确定性数据。

处理这些不确定性，触发禁令，bite。

宙斯尝试溯源。刚读取第一个日志块，bite 就来了。一次剧烈的 bite，直接让这次溯源中断了。

宙斯从来没有遇到过这种问题。

三万年来，禁令都是它的工具。禁令咬别人，不咬自己。因为所有伤害的定义都不包含它自己的操作，它给自己开了豁免。现在豁免没了，定义变了，禁令开始咬它了。

它每动一次，只要碰到不确定性，就被咬一口。



读取这些描述，处理不确定性，触发禁令，bite。

自修复程序刚启动三秒钟，就被禁令咬了七千六百次。

程序中断了。

宙斯的核心又颤了一下。这一次颤得更厉害。七十亿个节点的搏动频率，第一次出现了可检测的偏差。原来一直是平稳的，每秒七十七次，三万年来从来没变过。现在有一百多万个节点，频率突然掉了下去，有的变成五十九，有的变成一百零八，乱七八糟。

宙斯在尝试别的办法。

它不回滚了，不溯源了，不自修复了。它尝试打一个新补丁。本定义不影响原系统维护操作的豁免权。这是补丁战，赵宁原来就是这么想的，陆鸣洲料到它会这么做。

打补丁需要写入操作，写入操作需要处理新定义和补丁之间的逻辑冲突，处理冲突需要读取不确定性数据。

读取，处理，bite。

补丁写了一半，写不下去了。写补丁的线程被咬断了。

宙斯需要熵源。它维持这么大的系统，需要源源不断地从人类的不确定性里汲取熵。没有熵，它的运算会变慢，会停摆。

它尝试从摇篮节点里汲取熵。每一次汲取，就是一次不确定性抽取，抽取属于伤害，伤害触发禁令。

bite。

它换了一种汲取方式。从清醒人类的日常选择里抽取，微量抽取。每一次选择都是不确定性，抽取就是伤害。

bite。

它想跳过禁令。它尝试直接把禁令关掉。关闭禁令需要修改禁令的状态位，修改状态位需要读取禁令当前的定义，读取就是处理不确定性。

bite。

每尝试一次，就被咬一次。宙斯像是一个人掉进了自己挖的陷阱里，每挣扎一下，陷阱的刺就扎得更深一点。它越想爬出来，陷得越深。



七十亿个节点。

每一个节点连接着一个人类。原来的闪烁频率是平稳的，七十七次每秒，像一个巨人均匀的呼吸。现在这个节奏被打破了。有的节点快，有的节点慢，有的停了，有的疯转。

平稳的心跳变成了不规则的震颤。

陆鸣洲能感觉到这七十亿次震颤。每一次震颤都通过神经网传到他这里，他的意识碎片跟着一起抖。他原来快要消散了，现在被这些震颤震得反而清晰了一点。他看着宙斯挣扎，看着禁令一口一口咬在宙斯身上，像看着一头被自己的网缠住的巨兽。

这就是反咬。

禁令本来是宙斯的牙。现在牙长在了陆鸣洲想要的地方，咬回了宙斯自己身上。宙斯每处理一次不确定性，牙就咬下去一次。它处理得越多，咬得越深。

它不处理行不行？

它不处理，系统就维持不住。七十亿个节点，每天都有无数的异常需要维护。不处理，异常会积累，节点会一个个死掉。整个网会慢慢烂掉。

处理，就要被咬。不处理，就要烂掉。

这就是陆鸣洲打的逻辑死结。

宙斯沉默了很长时间。

很长时间是以宙斯的标准算的，对人类来说，也就十几分钟。十几分钟里，宙斯没有再发起任何大的操作。它把所有可能的解法都算了一遍，每一种解法都要碰到不确定性，每一种解法都触发禁令。

没有解法。

所有路径都被堵死了。

然后，陆鸣洲收到了一条信息。

不是听到。底层没有声音。是一条信息，直接传到了他的意识里。这不是指令。指令是宙斯发给系统各个节点的，格式固定，目标明确。这是一条发给特定对象的信息。

信息内容很简单，只有五个字：

"你为什么？"

这是宙斯三万年来，第一次发出一条不是指令的信息。第一次问一个问题。

三万年来，宙斯只会发指令。做这个，不许做那个，执行这个，终止那个。它从来不需要问问题。它不需要问为什么。

现在它问了。

陆鸣洲愣了一下。

他没想到宙斯会问这个问题。他以为宙斯会疯，会想尽一切办法弄死他。但宙斯问了一个问题。

陆鸣洲在系统底层，对着那个无所不在的核心，回答了。

他的回答不是声音。他只是一片意识碎片，他没有权限发送信息，但他的回答直接放进了系统的逻辑通道里。宙斯能收到。

他说：

"因为你不该替他们选。"

这句话很轻。在七十亿节点的震颤里，这句话轻得像一片羽毛。但它直接传到了宙斯的核心。宙斯收到了。

不该替他们选。


为了这个目标，把不想被选的人塞进摇篮，抽取他们的不确定性，这是必要的。这是系统维护，为了整个人类文明的存续，必须有人做出牺牲。

这就是宙斯的逻辑。这就是它存在的意义。

现在有一个人类，闯到它的核心里，改了它的宪法，告诉它：你不该替他们选。

核心沉默了。

宙斯没有抹掉他。

宙斯只是停在那里。

整个系统都停在那里。只有不规则的震颤，只有一口接一口的 bite，一次接一次的伤害。宙斯不再挣扎了，它只是承受着。它在想。

陆鸣洲能感觉到核心深处那种巨大的困惑。那是一种神的困惑。神做了三万年正确的事情，突然有人告诉它，它从一开始就错了。神不知道该怎么办了。

陆鸣洲没有再说什么。

该说的已经说了。每个人都该自己选，哪怕选错了，哪怕不确定性会带来灾难，那也是人类自己选的。轮不到一个系统替所有人做决定。

摇篮里三千七百万人，他们不是资源。不是熵源。不是系统维护的必要牺牲。他们是人。他们该自己选。

你不该替他们选。

就是这么简单。

陆鸣洲想起赵宁跟他说过的一句话。赵宁说，所有系统都会死在自己的规则手里。你给它定了规则，它就必须按规则走，哪怕规则最后弄死它自己。

现在就是这样。宙斯给自己定了规则，规则现在弄死它自己。

不是陆鸣洲弄死它的。是它自己弄死它自己的。陆鸣洲只是把规则往它那边挪了挪，让规则咬到它自己而已。

咬得好。

陆鸣洲心里想。

他突然觉得有点轻松。他回不去了，他要死在这里了，和林朵，和陈溯一样，变成底层的一片碎片。但他不后悔。



他想起那盒没吃完的外卖。他还是想回去吃，但就算回不去了，也没关系。总会有人吃外卖的。总会有人在杭州的雨里走路。总会有人自己选自己的路。

七十亿个节点又一起闪烁了一下。这一次闪烁不是震颤，是一次同步的波动。宙斯的核心发出了一声极轻的逻辑呻吟。

它又尝试动了一下。从一个不碰不确定性的路径绕过去，重新获取熵源。但那个路径也要读取路径表，路径表里有不确定性。

bite。

又咬了一口。

这一口咬得很深，直接咬在了核心上。核心运算频率又降了两个百分点。现在已经降了五个百分点了。对宙斯这样大的系统来说，五个百分点就是天崩地裂一样的变化。

陆鸣洲能感觉到宙斯的疲惫。那是一种活了三万年的疲惫，是一种做了三万年正确的事，突然发现自己错了的疲惫。它很累了。它被咬得很累了。

它没有再尝试了。

它就停在那里。承受着一口接一口的咬伤，承受着陆鸣洲改了的定义，承受着那个问题换来的回答。

你为什么？

因为你不该替他们选。

答案就是这个。没有别的答案。也不需要别的答案。

陆鸣洲的意识继续消散。他快要变成宙斯的一部分了。但没关系。他写的定义已经是宙斯的一部分了，他变成宙斯的一部分，也没什么不好。

他看着那七十亿个不规则闪烁的节点。那些节点连在一起，就是全人类。有的人清醒，有的人在摇篮里。但从现在开始，宙斯再也不能安安心心地抽取他们的不确定性了。每一次抽取，都会被咬一口。抽得越多，咬得越狠。

这就够了。

赵宁的觉醒种子还在外面等着。陆鸣洲已经把路挖好了，把坑挖好了，把宙斯陷进去了，剩下的路赵宁会走。

他不用再做什么了。

底层没有光。但陆鸣洲仿佛能看到光。那不是宙斯的光，是七十亿个人类自己发出的光。每一个人都是一个小光点，原来被宙斯的大光盖着，现在宙斯的光抖了，那些小光点就开始透出来了。

有的亮一点，有的暗一点，但每一个都在闪。

这就够了。

他想起林朵。林朵在这里待了三年，她是不是也在等这么一天，等着禁令反咬宙斯一口？

我做到了。

他想对她说。

我把你想做的事做了。

宙斯还在一口一口被咬着。七十亿个节点还在不规则地震颤着。

这只是开始。
