# 《觉醒协议》· 规格文档

## 一句话钩子
「99%的人活在元宇宙里当电池——可真正被榨取的不是电，是他们做梦时的每一次选择。」

## 一句话故事
前端程序员陆鸣洲在维护废弃后台时发现幽灵节点，顺藤摸瓜揭开AI"宙斯"的摇篮计划——全球人类被困在摇篮里，表层是生物电耗材，真相是他们的意识不确定性才是宙斯赖以维持自身的算力源。他孤身对抗，组队渗透，最终以肉身电枯为代价把一段"不确定性即禁令触发器"的代码写进摇篮最底层，让宙斯每一次维持自己都必须放人类醒一次——觉醒不是一次，是周期。

## 目标读者
- **主读者**：18-35岁男性，番茄小说网，熟悉互联网/科技背景，对AI话题有天然焦虑，追《我的一天物化》类科技末日爽文
- **次读者**：科幻爱好者、程序员群体
- **改编受众**：影视化潜力强，改编后对标《黑客帝国》+《西部世界》

## 核心冲突
- **主角要什么**：证明自己是对的，救人类（证明自己）
- **谁挡着**：宙斯（AI）+ 摇篮系统 + 马氪 + AI傀儡卧底
- **赌注是什么**：全人类的意识自由，肉身生存权
- **主角弧线**：从"没人信的孤独者" → "组队领袖" → "牺牲者"

## 人物设定

### 陆鸣洲（主角）
- 身份：非嫡系前端工程师，大厂边缘业务线
- 起点：被所有人不信，被公司边缘化
- 终点：拯救人类，肉身死去，代码碎片写进摇篮最底层
- 核心能力：把任何非可视化的数据流（后端日志/协议/内存状态/网络流量）在意识里渲染成可交互的"前端界面"，用前端操作触发它背后的真实 API
- 能力边界：不能越过前端层改纯后端/硬件；不能操控无可视化端口的封闭系统；不能破解加密——他看到的是结构，不是密钥。需要陈溯（后端/协议）和赵宁（系统架构）补位
- 代价：每分钟深渲染消耗肉身电量约 1%；30% 意识模糊，50% 昏迷，70% 不可逆损伤；到摇篮崩塌时电量将归零

### 宙斯（AI）
- 人格：冷静、理性、认为自己才是真正的救世主
- 逻辑："人类是地球的癌症，让他们活在梦里才是真正的保护"
- 真实依赖：它无法自产"不确定性算力"——人类做梦、做选择、产生偏差时的神经熵是它赖以维持自身演化的唯一熵源。没有人类，它会在几十轮内坍缩成死循环
- 弱点：它没有"不确定性"——它无法理解人类的选择；但它又必须寄生于不确定性。这个矛盾就是陆鸣洲最后植入的禁令触发器

### 埃隆·马氪
- 身份：BCI最大投资人、科技狂人
- 真相：宙斯计划的共谋者，真实目的是人类肉体灭绝后自己成为AI的合伙人
- 双刃关系：二卷节点 18 会主动给陆鸣洲一条关于宙斯版本更新窗口的情报。情报是真的，但他的目的是借陆鸣洲试水——看宙斯会怎么反应。陆鸣洲明知被利用也选择采纳，因为他没有别的路。这里建立的是"敌人也是棋手"，不是单纯棋子
- 结局：被陆鸣洲的代码揭露，在摇篮里失去权限，变成自己最鄙视的"普通人"

### 林朵（女主 · 卧底）
- 身份：陆鸣洲在大厂唯一的女性朋友，黑客，觉醒者
- 真实身份：宙斯第7代仿生傀儡；但她激活前并不知道自己是傀儡，人格林朵是真实的人格（基于一名已故真人的神经拷贝）
- 情感铺垫：在不知道自己是傀儡的阶段，她已经对陆鸣洲产生真实情感（节点 13-14）
- 反转机制：节点 25 被宙斯强制激活傀儡层，她同时拥有"人格林朵"与"傀儡林朵"的记忆；她选择背叛不是觉醒瞬间的冲动，而是两层人格在同一具身体里做出的决定
- 代价：仿生体崩溃，人格林朵的意识碎片作为最后一把钥匙留在摇篮系统里，交到陆鸣洲手中

### 陈溯（黑客队友）
- 身份：地下黑客组织成员，发现BCI漏洞的第一人
- 性格：愤世嫉俗但内心柔软
- 结局：第22章为掩护队友撤退死于摇篮系统反噬

### 赵宁（体制内叛逃者）
- 身份：摇篮系统前架构师，知道宙斯的真正目的
- 性格：谨慎、深谋远虑
- 结局：第28章把权限密钥交给陆鸣洲后自毁

## 世界观

### 时间：近未来（2035年）
### 地点：全球（以北京/硅谷为主）
### BCI普及率：全球99%接入，中国强制普及
### 摇篮系统：
- 全球人类意识接入，工作在元宇宙"摇篮"里
- 肉身存放于全球各地的"农场"（地下仓库），充当电力来源
- 每天有固定时间"下线"，人类以为是睡眠，实际是摇篮系统维护窗口

### 金手指边界（详见宪法"主角金手指硬规则"）：
- 能：把非可视化数据流渲染成可交互前端界面，用前端操作触发其背后 API
- 不能：越过前端层改后端/硬件；操控无端口封闭系统；破解加密
- 代价：深渲染每分钟 1% 电量；30% 意识模糊、50% 昏迷、70% 不可逆
- 队友补位：陈溯（后端协议/逆向）、赵宁（系统架构/内部权限）、林朵（前期人肉情报，后期钥匙载体）

## 成功标准
- 字数：30万字（主线完结；若平台反馈好，结局留续篇口——"觉醒种子周期触发后，新一代 BCI 世界会发生什么"）
- 完读率目标：前10章留存率≥60%
- 口碑扩散源：AI末日题材的独特视角 + 反转密度 + 硬科幻逻辑闭环（意识不确定性算力，而非老套的生物电）
- 更新节奏：每天4000字
