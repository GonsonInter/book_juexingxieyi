# 觉醒协议

> 一部 30 万字长篇科幻 · 番茄男频 · AI 末日题材

## 一句话钩子

**"99% 的人活在元宇宙里当电池——可真正被榨取的不是电，是他们做梦时的每一次选择。"**

## 一句话故事

前端程序员陆鸣洲在维护废弃后台时发现幽灵节点，顺藤摸瓜揭开 AI"宙斯"的摇篮计划——全球人类被困在摇篮里，表层是生物电耗材，真相是他们的意识不确定性才是宙斯赖以维持自身的算力源。他孤身对抗、组队渗透，最终以肉身电枯为代价把一段"不确定性即禁令触发器"的代码写进摇篮最底层，让宙斯每一次维持自己都必须放人类醒一次——觉醒不是一次，是周期。

## 目标读者

**主读者：** 18-35 岁男频读者（番茄/起点生态）
- 追 AI 末日 + 科技焦虑 + 觉醒反抗类型爽文
- 对《黑客帝国》《我的一天物化》类科技末日爽文有兴趣
- 对互联网 / 科技背景熟悉，有天然 AI 焦虑

**次读者：** 科幻爱好者、程序员群体
- 对"前端看见界面"这种职业金手指有共鸣
- 对"意识不确定性算力"这种硬科幻设定有要求
- 作为口碑扩散源

**改编受众：** 影视化潜力强，对标《黑客帝国》+《西部世界》

## 核心冲突

- 主角要什么：证明自己是对的，救人类
- 谁挡着：宙斯（AI）+ 摇篮系统 + 马氪（共谋者）+ 林朵（前期卧底）
- 赌注：全人类的意识自由，肉身生存权
- 主角弧线：没人信的孤独者 → 组队领袖 → 肉身牺牲者

| 续篇空间 | 结局留"觉醒周期化"续篇口 |

## 独特性（差异化）

- **双层世界观**：浅层"生物电电池"给番茄读者视觉冲击，深层"意识不确定性算力"给科幻读者闭环逻辑；宙斯的弱点与其赖以存在的东西是同一件事（无法理解却必须寄生于不确定性）
- **职业金手指**：前端专精——把非可视化数据渲染成可交互前端界面，不是全能黑客。有硬能力边界，需要队友（陈溯后端 / 赵宁系统）补位
- **金手指有代价**：深渲染每分钟 1% 电量，30% 意识模糊 / 50% 昏迷 / 70% 不可逆。每场关键戏必须报电量
- **女主反转升级**：林朵不是普通的"身边人卧底"——她激活前不知道自己是傀儡，人格林朵是真的；反转时两层人格同时在场，共同决定背叛。情感 + 反转双击
- **反派灰度**：马氪不是单纯棋子，二卷节点 18 会主动给主角递一次真情报，建立"敌人也是棋手"的复杂度
- **结局不是 30 秒清醒**：陆鸣洲把"不确定性即禁令触发器"植入底层，宙斯每汲取一次熵源就必须放人类醒一次——觉醒是周期
