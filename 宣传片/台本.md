# 《觉醒协议》宣传片台本 · 4 × 15s

## 整体说明

- **总时长**：60s（4 段 × 15s），每段独立成立、可单条投放
- **节奏**：每段一个独立钩子，首三秒必须抓住划走的手
- **画面基调**：冷白 / 暗红 / 数据流灰；后颈贴片浅蓝光是视觉锚
- **字数节制**：每段画面字幕≤30 字，旁白≤45 字
- **底部水印**（贯穿四段）：右下角小字 "《觉醒协议》| 已完结 100 章 | 全网同步"

---

**目标**：让看客以为是日常 vlog，最后两秒翻面成科幻。

|---|---|---|---|

## 投放顺序建议

- **D1**：E1（99.4%）—— 拉新，世界观钩子
- **D2**：E4（30 秒）—— 情绪钩子，跳过中间剧透，吸引"想看完结局"的人
- **D3**：E2（78 亿）—— 价值观钩子，靠 slogan 出圈
- **D4**：E3（三处节点）—— 留给已经被前三条种草的用户，看完直接转化

> 不按 1→4 顺序投是因为：E1 适合冷启动，E4 出情绪，E2 出 slogan 二次传播，E3 是给已有兴趣的人看的"高燃集锦"，转化效率最高。

## Seedance 操作清单

每段跑片时按这个顺序：

1. **打开即梦 → 视频生成 → Seedance 模型 → 选 15s**
2. **挂参考图**（按当段 prompt 里 @标记的素材，从 `素材/` 目录上传）
   - 人物参考槽 → 主角定妆 / 林朵 / 赵宁等
   - 首帧槽 → 当段对应的开场图（如 E1 用 08-主角工作站，E2 用 05-营养舱阵列）
   - 风格参考 → 当段需要锁定风格的图
3. **粘贴 prompt** → 跑生成
4. **每段抽 3-4 次** → 挑最满意的一版
5. **导出**：1080p 30fps，命名为 `E1.mp4` / `E2.mp4` / `E3.mp4` / `E4.mp4`
