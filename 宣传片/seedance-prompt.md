# Seedance Prompt · 4 段拼接版（看图复刻 + 真实动画 · 一次成片）

## 🔑 核心原则

**每天只有 1 次 Seedance 机会**，所有文字内容、场景细节、特效已经在素材图里固化。

- **70% prompt = "严格复刻 @图N"**（让 AI 围绕图扩展）
- **20% = 真实人物/环境动作**（不是图片缩放，是真有人在动）
- **10% = 声音/字幕**

> 🚫 **铁律 1**：所有出现在素材图里的文字（屏幕红字、腕表数字、全息广告、终极标题），**严禁让 AI 改写**。所有出现在素材图里的人物面部/装扮/特效（电路纹路、意识光粒子、手部细节），**严禁让 AI 简化**。
>
> 🚫 **铁律 2（关键 · 防 AI 乱画字）**：**画面里严禁出现任何 prompt 未明确指定的文字、汉字、英文、数字、字符、广告、招牌、标识、贴纸、纹身、印记、水印、商标、Logo、车站名、地名、海报、报纸、霓虹文字、装饰字符**。AI 不允许在画面里"自由发挥"添加任何文字装饰元素——主角脸上、衣服上、帽子上、皮肤上、墙上、车厢内、街边、招牌上、屏幕背景等所有可能位置都**禁止出现 prompt 里没有明确指定的字符**。素材图里已有的文字（@图2 屏幕红字 / @图3 腕表 47% / @图1 全息广告 / @图3 觉醒协议标题）必须严格保留**原样**不能改写、不能添加、不能误植到其他位置。**只有 prompt 在"画面文字一览"或"金句砸落"段落明确指定的字幕才能在画面里出现**，其他一切 AI 自动生成的文字（包括"千"、"于"、"无"、"卐"等装饰汉字、对称图案、八卦图、车站名牌）都被禁止。
>
> 🎬 **铁律 3**：每个镜头必须有**真实人物动作或环境动作**——不是镜头缩放推拉，是**真有人在动、真有事在发生**。这不是 PPT 动画，是电影预告片。每镜 3-5 个明确可执行的动作节奏，让画面有生命。

## 🎬 整体结构

**4 段连续拼接**，共享一条 60 秒张力曲线，调性参考《沙丘》《奥本海默》《信条》《银翼杀手 2049》预告片。
