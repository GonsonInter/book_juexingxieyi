# Seedance Prompt · 4 段拼接版（看图复刻 + 真实动画 · 一次成片）

## 🔑 核心原则

**每天只有 1 次 Seedance 机会**，所有文字内容、场景细节、特效已经在素材图里固化。

- **50% prompt = "以 @图N 为视觉锚点"**（人物面孔/服化/已固化文字必须保留，但镜头机位、景别、运动允许变化——**不是把素材图当首帧死锁**）
- **20% = 摄影机运动**（handheld / dolly / tracking / rack focus / whip pan / crane——**镜头本身在动**，不是只有主体在动）
- **20% = 真实人物/环境动作**（演员在动 + 环境运动层并发：飘尘 / 雨丝 / 蒸汽 / 雾气 / 灯光巡转 / 电缆摆动）
- **10% = 声音/字幕**

> 🚫 **铁律 1**：所有出现在素材图里的文字（屏幕红字、腕表数字、全息广告、终极标题），**严禁让 AI 改写**。所有出现在素材图里的人物面部/装扮/特效（电路纹路、意识光粒子、手部细节），**严禁让 AI 简化**。
>
> 🚫 **铁律 2（关键 · 防 AI 乱画字）**：**画面里严禁出现任何 prompt 未明确指定的文字、汉字、英文、数字、字符、广告、招牌、标识、贴纸、纹身、印记、水印、商标、Logo、车站名、地名、海报、报纸、霓虹文字、装饰字符**。AI 不允许在画面里"自由发挥"添加任何文字装饰元素——主角脸上、衣服上、帽子上、皮肤上、墙上、车厢内、街边、招牌上、屏幕背景等所有可能位置都**禁止出现 prompt 里没有明确指定的字符**。素材图里已有的文字（@图2 屏幕红字 / @图3 腕表 47% / @图1 全息广告 / @图3 觉醒协议标题）必须严格保留**原样**不能改写、不能添加、不能误植到其他位置。**只有 prompt 在"画面文字一览"或"金句砸落"段落明确指定的字幕才能在画面里出现**，其他一切 AI 自动生成的文字（包括"千"、"于"、"无"、"卐"等装饰汉字、对称图案、八卦图、车站名牌）都被禁止。
>
> 🎬 **铁律 3**：每个镜头必须有**真实人物动作或环境动作**——是**真有人在动、真有事在发生**。这不是 PPT 动画，是电影预告片。每镜 3-5 个明确可执行的动作节奏，让画面有生命。
>
> 🎥 **铁律 4（防 PPT 感 · 关键）**：每个镜头**必须有明确的摄影机运动**（handheld shake / dolly-in / tracking / rack focus / whip pan / crane 至少一种）+ **必须有持续的环境运动层**（雾气 / 飘尘 / 雨丝 / 蒸汽 / 灯光巡转 / 电缆摆动 / 烟流 / 浮粒 至少一种）。**禁止纯静态机位 + 主体微动**模式（这就是 PPT 感的根源）。**禁止段内"黑场过渡 0.3 秒"切镜**——同一段内镜头切换必须用 match cut（动作/光色/形状衔接）/ whip pan（甩镜）/ 硬切（hard cut）；黑场**只允许出现在每段段尾金句段**。**铁律 1 锁文字 + 铁律 4 放运镜**：AI 应把素材图理解为"演员面孔/服化道/已固化文字"的锚点，**不是首帧静态构图**——镜头角度、景别、运动方向在保留人物/场景特征的前提下允许变化。

## 🎬 整体结构

**4 段连续拼接**，共享一条 60 秒张力曲线，调性参考《沙丘》《奥本海默》《信条》《银翼杀手 2049》预告片。

## 📐 四段衔接接力（关键 · 防止段间断裂感）

**四段是 4 次独立 Seedance 生成（每段 15s）后剪映拼接**，所以每段的**首帧 ↔ 前段末帧**、**末帧 ↔ 下段首帧**必须像接力棒一样能扣上。AI 经常在每段开头"重启沉默仪式"——这会让拼出来的 60s 像 4 个独立短片串烧。**段首 0-1.5s 的开场不要加摄影机运动**（保持低调延续），**段尾金句段也不要加运镜**（留干净黑场给下段）。运镜只在每段中段的 3 个主体镜头里施展。

| 接缝 | 前段末帧（最后 1 秒画面/音） | 下段首帧（第 0-1 秒画面/音） | 衔接钩子 |
|---|---|---|---|
| **E1 → E2** | 纯黑画面 + 低频"咚——"长 reverb 余响 + 完全静默 | **黑场延续不重启** + 低频"咚"余响**未消散** + 第 0.3 秒下方浮现"B3."白字 | "咚"余响 + 纯黑画面**贯穿过段，不要 fade out 再 fade in** |
| **E2 → E3** | 纯黑画面 + 红线扫过短促警报"嘟——"戛然而止 + 低频"咚"长 reverb | **黑场延续** + "咚"余响**未消散** + 第 0.2 秒画面中央"47:00"红色数字浮入（静态非倒数） | 红色作为色彩接力——E2 末尾红线 → E3 开场红色数字，红色作为衔接锚点 |
| **E3 → E4** | 纯黑画面 + 三声"咚——"逐次砸落后第 3 声长 reverb 余响 | **黑场延续 1.5 秒**（不要纯黑 2s 重启）+ 第 3 声"咚"余响**未消散** + 后期叠加金句"每个周期，所有人会醒 30 秒。" | 黑场期不渲染中文字幕（金句完全后期叠加），AI 只生成纯黑+延续余响 |
| **E4 → 结束** | 终极标题"觉醒协议 AWAKENING PROTOCOL"缓慢淡出至纯黑 + 极长 reverb 钟声尾音（≥3 秒） | — | 整支唯一一次出标题，淡出而非硬切 |

**衔接铁律**：
- 每段**开场 0-1.5s 不要加摄影机运动**（保持低调延续，让前段"咚"余响有空间舒展）
- 每段**段尾金句段不要加运镜**（留干净黑场和音效给下段接力）
- 每段**首帧的"开场字幕"（"2035." / "B3." / "47:00" / 金句）必须是浮现到位**（不是从前段画面残影里渐变出来）
- 每段**末帧的纯黑**必须是真实纯黑（RGB 0,0,0），不要带任何残影/光晕，让剪映拼接时段间无缝
- 4 段的低频"咚"音效**必须使用同一个合成器音色**（4 段 prompt 都写"BRAAAM 风深沉低频铜管脉冲"），让接力时音色一致

## E1 · 47%（开场段 · 沉默仪式 + 宙斯定罪 + 真实动画）

**首帧**：02-主角工作站

```
电影预告片风格，参考《沙丘》《奥本海默》《信条》《银翼杀手 2049》预告片调性。这是一支 60 秒大片预告片的第 1/4 段。时长 15 秒，2.39:1 宽银幕，photorealistic 真实质感。整段以"以参考图为人物/服化/已固化文字的锚点 + 镜头主动运动 + 真实人物/环境动作扩展"为核心原则——不是图片缩放、不是静态机位+主体微动，**镜头本身在动 + 演员在动 + 环境在动**三层并发。每镜 3-5 个明确动作节奏。**段内禁止"黑场过渡"切镜**，镜头切换用 match cut / whip pan / 硬切。视频包含原生音频：仅环境音+具体音效+人声 VO+字幕音，严格不包含任何背景音乐、旋律、配乐、歌词。

【0-1s · 沉默开场】
画面：纯漆黑 0.6 秒，画面正中浮现一行白色小字"2035."保持 0.3 秒，缓慢消散，回归漆黑 0.1 秒。
声音：极轻低频电流嗡鸣（约 -28dB）+完全寂静感。

【1-4s · 第一镜：腕表特写（@图3 锚定 + 真实动作 + 镜头运动）】
📷 摄影机：手持微震（handheld shake，幅度 0.5°，呼吸节奏的轻颠）+ 极缓 push-in 1 秒至腕表 LED 数字。开场用极快淡入（0.2 秒，不是慢淡入），让镜头一进入就在动。
🌫️ 环境层：腕表周围空气中可见极细灰尘颗粒在冷蓝光中悬浮飘移 + 表盘玻璃表面有极轻微的反光涟漪（呼应手腕颤抖） + 后景虚焦中有不规则光点闪烁（暗示工位屏幕余光）。
画面：以 @图3 为视觉锚点——黑色圆形腕表在左手腕上，LED 显示红色"47%"+下方白字"BATTERY"+红色环形进度条（半圆亮）。**严禁更改 @图3 的文字、数字、配色**，但**镜头角度允许从手腕侧前方略微变化**（不是死锁正视图）。
- **第 1.5 秒：手腕极轻地翻转约 30°**（让腕表角度变化），LED 红光闪烁频率从慢闪变急促
- **第 2.3 秒：另一只手（主角的右手）从画面外伸入画面**，食指指尖紧张地碰到腕表玻璃表面（暗示在确认数字）
- **第 3 秒：右手指尖收回离开画面**，腕表数字红光骤然变亮闪一下（警告级闪烁）
- **第 3.5 秒：手腕轻微颤抖一下**（主角紧张反应）
声音：腕表"嘀——"低沉一声+心跳脉冲底噪（每 1.5 秒一次）+第 2.3 秒手指碰到玻璃时一声极轻的"嗒"+一声深沉低频铜管脉冲（合成器音效，非配乐非旋律）从远处涌来又退去。

【4-8s · 第二镜：屏幕异常（@图2 锚定 + 主角真实反应 + 镜头运动）】
📷 摄影机：**无黑场硬切**（match cut：腕表的红光闪烁 → 屏幕的红字闪烁，红光颜色匹配剪辑直接接上）。本镜采用低角度斜后方仰拍主角后脑+屏幕，slow dolly-in 1 秒推进至屏幕红字最大化；第 6.8 秒主角缓慢后靠瞬间镜头被动后撤同步（同步反应）+ rack focus（焦点切换）从屏幕红字 racking 到主角侧脸表情。
🌫️ 环境层：屏幕散发的红光在桌面飘浮的尘埃颗粒上有反射（如同放映机光束里漂浮的灰尘） + 后景吊灯/天花管线在画外光源下极轻摇曳 + 桌面咖啡杯液面因主角缓慢后靠时桌子微震有极细同心波纹 + 主屏的红色字符流持续从右向左滚动。
画面：以 @图2 为视觉锚点——主角侧脸 3/4 侧身坐在工作站前，左屏红色字符"NODES ONLINE: 7,830,000,000"+下行"GHOST NODE DETECTED"+滚动红字符流，右屏青绿色节点关系图，桌上咖啡杯+电缆。**严禁更改屏幕文字内容**，但**镜头机位允许是低角度斜后方而非正面**。
- **第 4.5 秒：主角的右手缓慢伸向键盘**+敲击 1 个键，屏幕红字闪烁一下（系统响应）
- **第 5.5 秒：他的眉头几不可察地锁紧 1mm**（职业 analyst 看到异常数据时的凝固，**不是惊愕、不是瞠目**——表情几乎不变，只有眉间微锁 + 视线在屏幕红字上定住不动）
- **第 6 秒：他的喉结上下滑动一次**（极轻微的吞咽，克制）
- **第 6.8 秒：他缓慢后靠半寸**（不是猛地后仰，是身体不自觉绷紧后的微微远离），椅背轻微吱嘎一声，**面部光线从红光主导渐变为冷蓝光主导**
- **第 7.5 秒：他的嘴唇极轻地抿了一下**（屏息思考），呼吸放慢但加深，胸口起伏极克制
```
